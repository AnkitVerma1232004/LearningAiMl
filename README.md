# Learning AI/ML - Python & Foundations

Welcome to my learning journey into Artificial Intelligence and Machine Learning. This repository serves as a central hub for my progress, covering Python fundamentals, Object-Oriented Programming (OOP), and practical assignments.

## 📂 Repository Structure

- **Assignments/**: Solutions to modular Python assignments covering basics to advanced logic.
- **codes/**: Jupyter notebooks (`.ipynb`) detailing core concepts:
    - `python_basics`: Control flow, loops, and functions.
    - `oops_concepts`: Classes, Inheritance, Encapsulation, and Polymorphism.
- **makelifeeasycode/**: Automation scripts for workflow optimization (e.g., folder creation).
- **notes/**: Supplemental PDF resources for quick reference.

Mastered basic syntax, data types, and logical operations.
